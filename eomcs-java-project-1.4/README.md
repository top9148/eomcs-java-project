## 프로젝트 - 수업관리 시스템  

### 과제 1: 한 개의 수업 내용을 입력 받아 출력하라.

- App.java
    - main()에 코드를 작성한다.

실행 결과

```
번호? 1
수업명? 자바 프로젝트 실습
수업내용? 자바 프로젝트를 통한 자바 언어 활용법 익히기
시작일? 2019-01-02
종료일? 2019-05-28
총수업시간? 1000
일수업시간? 8

번호: 1
수업명: 자바 프로젝트 실습
수업내용: 자바 프로젝트를 통한 자바 언어 활용법 익히기
기간: 2019-01-02 ~ 2019-05-28
총수업시간: 1000 시간
일수업시간: 8 시간
```
